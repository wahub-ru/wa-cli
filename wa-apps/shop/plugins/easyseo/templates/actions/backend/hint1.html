Метаданные

  {$store_info.name} - название магазина
  {$store_info.phone} - телефон магазина
  {$storefront.name} - название витрины
  {$host} - текущий домен
  {$wa_app_url} – ссылка на витрину
  
  
Категории

  {$category.name} - название категории
  {$category.seo_name} - SEO-название категории
  {$root_category.name} - название корневой категории
  {$parent_category.name} - название родительской категории
  {$parent_categories} - массив родительских категорий
  {$parent_categories_names|sep:\' \'} - путь к категории через пробел
  {$category.products_count} - количество товаров в категории
  {$category.min_price} - минимальная цена в категории
  {$category.max_price} - максимальная цена в категории
  {$category.fields[1].value} - список кастомных полей у отдельных категорий
  {$wa->shop->categoryUrl($category)} – ссылка на категорию
  {$wa->shop->categoryUrl($root_category)} – ссылка на родительскую категорию
  
  
Пагинация

  {$page_number} - номер страницы
  {$pages_count} - количество страниц
  
  
Товары

  {$product.name} - название товара
  {$product.seo_name} - SEO-название товара
  {$product.format_price} - цена товара
  {$product.price} - текущая цена товара
  {$product.sku} - артикул товара
  {$categories_names|sep:\' \'} - путь к товару через пробел
  
  
Бренды

  {$brands} - список брендов
  {$brand_names} - список наименований брендов
  {$brand_names|randomize|array_slice:0:3|sep:\', \'} - список из 3х случайных наименований брендов через запятую
  
  
Определенный бренд

  {$brand.name} - наименование бренда
